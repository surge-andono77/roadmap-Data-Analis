ROADMAP DATA ANALYST — ENTERPRISE GRADE
1. FUNDAMENTALS ( pondasi wajib )
  Outcome: bisa membaca data, memahami konteks bisnis, dan siap masuk workflow analytics.
    
  1.1. Business & Analytical Mindset
    * Critical thinking: problem → hypothesis → data → insight.
    * KPI & Metrics: revenue, churn, retention, CAC, LTV, conversion funnel.
    * Domain knowledge: sales, marketing, finance, operations.
      
  1.2. Spreadsheet Mastery (Excel/Google Sheets)
    * VLOOKUP/XLOOKUP, INDEX-MATCH
    * Pivot Table
    * Conditional formatting
    * Basic charts
  > Ini baseline skill semua Data Analyst.
      
2. SQL MASTERCLASS ( core skill 80% pekerjaan )
  Outcome: mampu query data skala enterprise di warehouse (BigQuery, Snowflake, Redshift).

  2.1. SQL Dasar
    * SELECT, WHERE, ORDER BY
    * LIMIT, DISTINCT
  
  2.2. SQL Menengah
    * JOIN (inner, left, right, full)
    * GROUP BY + Aggregate
  
  2.3. SQL Advanced
    * Window Function (RANK, ROW_NUMBER, LAG, LEAD)
    * CTE & Subquery
    * Pivot/Unpivot
    * Performance tuning (index, clustering)
  > *SQL adalah skill paling krusial — tidak bisa dinegosiasikan.*

3. DATA MANIPULATION (Python)
  Outcome: bisa bersih-bersih data, transformasi, dan automasi pipeline kecil.
  3.1. Python Dasar
    * Variables, condition, loops
    * Functions
    * Error handling
    
  3.2. Data Analysis Libraries
    NumPy: array operations
    Pandas:
      * read_csv(), groupby(), merge(), pivot
      * handling missing values
      * feature engineering

4. EDA — Exploratory Data Analysis
  Outcome: dari dataset raw → insight yang actionable.
    Step:
      * Univariate analysis
      * Bivariate/multivariate
      * Data profiling
      * Correlation analysis
      Tools: Pandas, Matplotlib, Seaborn.

5. DATA VISUALIZATION & STORYTELLING
  Outcome: bisa menjual insight ke manajemen secara powerful.
    Tools (wajib pilih salah satu, idealnya dua):
      * Power BI (enterprise corporate)
      * Tableau (analytics heavy)
      * Looker Studio (entry level)
    Skill:
      * Build dashboard end-to-end
      * KPI tracking
      * Drill-through & filters
      * Design principle (UX, color, hierarchy)

6. STATISTICS FOR ANALYTICS
  Outcome: bisa validasi insight secara kuantitatif.
    Materi inti:    
      * Descriptive statistics
      * Probability
      * Distribution
      * Hypothesis testing: t-test, chi-square
      * A/B Testing
      * Confidence interval
    > Fokus ke statistik terapan, bukan teori.

7. BUSINESS ANALYTICS (mid-level skill)
  Outcome: bisa menjawab pertanyaan bisnis dengan model analitis.
    Use cases:
      * Cohort & retention
      * Segmentation
      * Funnel analysis
      * Forecasting sederhana
      * Revenue analysis
      * Customer behavior analytics

8. DATA WAREHOUSE & BI PIPELINE
  Outcome: paham cara data mengalir dari source → warehouse → dashboard.
    Pelajari:
      * ETL/ELT
      * Star schema (facts + dimensions)
      * Data orchestration (Airflow, Prefect)
      * Cloud warehouse (BigQuery/Snowflake/Redshift)

9. PRODUCTIVITY & ENTERPRISE SKILLS
  Outcome: siap kerja tanpa babysitting.
    Skill tambahan:
      * Git (version control)
      * Jira/Confluence (workflow)
      * Stakeholder management
      * Requirement gathering
      * Email professionalism

10. PORTFOLIO BUILDING (wajib)
   Minimal 5 project portfolio:
    1. SQL analytics case
    2. Python EDA + data cleaning
    3. Dashboard (Power BI / Tableau)
    4. Cohort analysis
    5. End-to-end mini pipeline
       (ETL → Warehouse → Dashboard)
